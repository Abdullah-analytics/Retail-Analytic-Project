# Retail Analytics Project

## Description
This project analyzes sales data of an online retail store from **2009 to 2011** to uncover trends, understand customer behavior, and generate actionable insights.  
The main focus is on using **PostgreSQL** for data extraction and analysis, and **Power BI** for interactive visualizations. **Python** is used minimally for small data manipulations.

---

## Current Status
- Project has just started
- Dataset imported into PostgreSQL
- Initial data exploration in progress in Power BI

---

## Objectives
- Identify best-selling products
- Analyze customer purchase patterns
- Explore sales trends over time
- Detect cancellations and returns
- Create dashboards and visualizations for actionable insights

---

## Tools & Technologies
- **PostgreSQL** – For data storage and querying
- **Power BI** – For interactive dashboards and visualizations
- **Python (pandas)** – For small-scale data manipulations

---

## Dataset
- Online Retail Sales Data (2009–2011)
- Contains customer orders, product details, quantities, prices, and country of purchase
- Imported into PostgreSQL for analysis

---

## Next Steps
- Clean and preprocess the dataset in PostgreSQL
- Build queries to extract key metrics:
  - Total revenue per customer and product
  - Top-selling products and countries
  - Cancellations and return rates
  - Seasonal trends and peak order times
- Create interactive dashboards in Power BI
- Summarize insights and recommendations for business decisions
