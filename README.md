# Retail Analytics Project

## Description
This project analyzes sales data of an online retail store to uncover trends, customer behavior, and insights.  
The main focus is on using **PostgreSQL** for data extraction and analysis, and **Power BI** for interactive visualizations. Python is used minimally for small data manipulations.

## Current Status
- Project just started  
- Dataset imported into PostgreSQL  
- Initial data exploration in progress in Power BI  

## Objectives
- Identify best-selling products  
- Analyze customer purchase patterns  
- Explore sales trends over time  
- Create dashboards and visualizations for actionable insights  

## Tools & Technologies
- PostgreSQL – for data storage and querying  
- Power BI – for interactive dashboards and visualizations  
- Python (pandas) – for data manipulations  

## Next Steps
1. Clean and preprocess the dataset in PostgreSQL  
2. Build queries to extract key metrics  
3. Create visualizations and dashboards in Power BI  
4. Summarize insights and recommendations
